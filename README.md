# -Custom-Checkbox-using
Projeto que reproduz um painel de configurações rápidas do sistema, com design limpo, caixa flutuante e interruptores funcionais. Simula o acesso a funções como rede sem fio, conexão sem fio e compartilhamento de internet.

## ✨ Funcionalidades
- Layout centralizado com fundo azul
- Caixa de configurações com cantos arredondados
- Opções: Wi‑Fi, Bluetooth e Ponto de Acesso Portátil
- Interruptores ligado/desligado com estilo visual
- Código simples e fácil de alterar

## 📂 Estrutura dos Arquivos
 
 
pasta_configuracoes/
├── index.html      → Estrutura da página
├── style.css       → Estilos e aparência
└── README.md       → Este arquivo
 
plaintext
  

## 🛠️ Tecnologias Utilizadas
- *HTML5*: estrutura dos elementos
- *CSS3*: cores, sombras, bordas e posicionamento

## 🚀 Como Usar
1. Salve os arquivos na mesma pasta
2. Abra o arquivo index.html em qualquer navegador
3. Altere cores, textos ou adicione novas opções no código

## 🎨 Personalização
- Mude a cor do fundo ou da caixa no arquivo style.css
- Adicione mais itens como “Som”, “Brilho”, “Modo Avião”
- Ajuste tamanhos e espaçamentos conforme preferir
